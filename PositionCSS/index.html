1) float

float controls whether an element is taken out of normal flow and “floats” to a side, allowing inline content (like text) to wrap around it.

Syntax

.box { float: left; }

Values

  • none
  • left
  • right
  • inline-start
  • inline-end

2) position

position controls how an element is placed and whether it stays in normal flow. Some values allow you to use top/right/bottom/left offsets.

Syntax

.box { position: absolute; top: 20px; left: 20px; }

Values

  • static
  • relative
  • absolute
  • fixed
  • sticky

Tip: absolute positions relative to the nearest ancestor that is not static. This demo sets the stage to position: relative.

3) flexbox

Flexbox is a layout system for one-dimensional layouts (row or column). You usually set display: flex on a container, then use alignment properties.

Core syntax

.container { display: flex; justify-content: center; align-items: center; }

Common flex values

  • display: flex | inline-flex
  • flex-direction: row | row-reverse | column | column-reverse
  • justify-content: flex-start | center | flex-end | space-between | space-around | space-evenly
  • align-items: stretch | flex-start | center | flex-end | baseline
  • flex-wrap: nowrap | wrap | wrap-reverse
