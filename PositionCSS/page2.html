1) grid

CSS Grid is a two-dimensional layout system (rows AND columns). You typically set display: grid on a container and define tracks with grid-template-columns and grid-template-rows.

Core syntax

.grid { display: grid; grid-template-columns: 1fr 1fr 1fr; gap: 12px; }

Common grid values

  • display: grid | inline-grid
  • grid-template-columns: repeat(n, 1fr) | 200px 1fr ...
  • justify-items: start | center | end | stretch
  • align-items: start | center | end | stretch
  • gap: length

2) vertical-align & text-align

text-align aligns inline content inside a block (like text inside a p). vertical-align mainly affects inline/inline-block/table-cell alignment. It does not vertically center block elements by itself.

Syntax

.text { text-align: center; }
.inline { vertical-align: middle; }

Values

  • text-align: left | right | center | justify | start | end
  • vertical-align: baseline | middle | top | bottom | text-top | text-bottom | sub | super

3) media queries

Media queries let CSS apply styles conditionally based on the device/viewport. Common features include max-width, min-width, and orientation.

Syntax examples

@media (max-width: 720px) { ... }
@media (min-width: 721px) and (max-width: 1100px) { ... }
