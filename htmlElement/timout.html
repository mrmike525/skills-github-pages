HTML Element Generator 1.0

This is a html Element Generator.

To proceed...

  1. Select an Element in the drop down menu
  2. Type your text in the text area
  3. Choose the font size with the slider, (font size only applies to paragraphs and lists)
  4. Select font color
  5. Select background color for the Element area.
  6. Check the box if you would like element created to be delayed by 3 seconds
  7. Click the create element button.






**Important Double Click on the element to remove that element**
**font size slider only applies to paragraphs and lists**
